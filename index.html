Live Real-time Translation Platform

Talk, Play n’ Health: The All-in-One Table Bot

以樹梅派為基底，製作桌上型機器人，提供使用者聊天、遊戲、健康檢測、物聯網、環境感測等功能。銜接AI大型語言模型，讓使用者可以進行日常對話、撥放音樂，也提供居家控制，開關風扇等。利用姿勢辨識模型，可以檢測使用者的坐姿狀態，並語音播報提醒；利用手勢辨識模型，可以進行猜拳互動。同時連接溫濕光度感測，顯示即時數值與燈號提示。

Spring 2025

程式語言課程專題 Programming Language Course Project

"OurScheme" 直譯器

“OurScheme” Interpreter

以C++語言撰寫，類Scheme語言直譯器。輸入OurScheme語法指令，系統可以輸出該指令的執行結果。從輸入的高階語言開始，進行詞彙切片、語法剖析、建立抽象語法樹，並執行像是四則運算、字串操作、條件判斷、定義函式、呼叫函式、區域變數綁定等指令功能。完整學習處理多層嵌套、遞迴高階函式的程式運作邏輯。

A Scheme-like interpreter written in C++. The system takes OurScheme syntax commands as input and outputs the execution results. It starts from high-level language input, performs lexical analysis, syntax parsing, builds an abstract syntax tree, and executes instructions such as arithmetic operations, string manipulations, conditional statements, function definitions, function calls, and local variable bindings. It fully supports handling multi-level nesting and recursive higher-order functions.

Spring 2024

計算機組織課程專題 Computer Organization Course Project

管線化CPU設計

Pipelined CPU Design

以Verilog實作五階管線化CPU，取指、指令解碼、執行、存取記憶體、回寫，實現16道MIPS指令。透過建構算術邏輯單元、控制單元、位移器、多工器、乘法器、計數器等元件，完成算術運算、邏輯運算、資料搬移與分支跳躍指令。搭配暫存器與控制訊號，同時利用無操作指令與危障控制，避免資料發生衝突，確保正確傳遞。

Implemented a five-stage pipelined CPU in Verilog, covering instruction fetch, instruction decode, execution, memory access, and write-back, supporting 16 MIPS instructions. By constructing components such as the arithmetic logic unit (ALU), control unit, shifter, multiplexer, multiplier, and counters, it performs arithmetic operations, logical operations, data transfer, and branch/jump instructions. With registers and control signals, it employs no-operation instructions and hazard control to prevent data conflicts and ensure correct data propagation.

Fall 2023

組合語言與嵌入式系統課程專題 Assembly Language and Embedded System Course Project

Julia Set

以ARM組合語言結合C語言實作Julia Set分形圖形繪製，程式透過雙層迴圈計算每個像素在複數平面上的對應值，並反覆套用迭代公式，決定像素顏色，最後輸出呈現完整影像。實際運用立即值、暫存器、位移三種指令格式，並應用條件執行與指定指令，同時面對有限暫存器下的函數呼叫、除法運算與變數配置挑戰，學習高低階語言的整合。

Implementing Julia Set fractal rendering by integrating ARM assembly with C. The program uses nested loops to calculate the corresponding value of each pixel in the complex plane, repeatedly applying the iterative formula to determine pixel colors, and finally outputs the complete image. In practice, it makes use of immediate, register, and shift instruction formats, while also applying conditional execution and specific instructions. At the same time, challenges such as function calls under limited registers, division operations, and variable allocation are addressed, providing hands-on experience in combining high- and low-level programming languages.

Spring 2023

運算思維與程式設計課程專題 Computational thinking and programming Course Project

過河問題

River-Crossing Problem

以Python語言實作八人過河問題，問題設定包含船隻僅能乘載兩人，僅部分成員能操控船，以及多項限制條件。藉由九位元布林串列表示船與人物位置，並排除違反規則的非法狀態，構建合法狀態圖，再以最短路徑演算法搜尋全員由東岸至西岸的最小步數解。此專案為第一次學習Python程式設計，並以模組化方式解決經典謎題。

Implemented the eight-person river-crossing problem in Python. The problem is defined such that the boat can carry at most two people, only some members are able to operate the boat, and multiple constraint rules apply. The boat and each person’s position are encoded as a 9-bit boolean list; illegal states that violate the rules are excluded and a graph of valid states is constructed. A shortest-path algorithm is then used to find the minimum-step solution that transfers everyone from the east bank to the west bank. This project was my first introduction to Python programming and tackles the classic puzzle using a modular design.

Spring 2025

陌生隨身碟使用行為與風險認知之探討：以中原大學為例

A Study on the Behavior and Risk Perception of Using Unknown USB Flash Drives

隨身碟實驗研究主持人。設計網頁、檔案與紀錄系統，觀察人們對於校園中陌生隨身的使用情形，並以資通安全及資訊素養的角度出發，研究受試者的動機與行為。實際於校園中投放62個隨身碟，約76% 被撿拾、移動，約13% 被開啟。發現受試者的開啟動機以好奇心為主，普遍存在風險認知但行動不足，且不同背景的人員皆有開啟行為。

Principal investigator of the USB flash drive experiment study. Designed the website, file system, and recording system to observe how people interact with unknown USB drives on campus, approaching the research from the perspectives of information security and digital literacy to study participants' motivations and behaviors. A total of 62 USB drives were deployed on campus, with approximately 76% picked up or moved and about 13% opened. Findings indicate that curiosity was the main motivation for opening the drives; participants generally had some risk awareness but insufficient action, and individuals from various backgrounds engaged in opening behaviors.
